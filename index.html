Track configurator

Vue 3 Composition Api, Vuex, Router

Ключевая мотивация

Помочь клиентам конфигурировать сложные трековые системы освещения, не вникая в технические детали и не выясняя какие комплектующие нужны для ее работы.

Стек

  • Vue 3 Composition Api
  • Vuex, Vue Router
  • i18n
  • Bootstrap, SCSS
  • Svgdotjs
  • vue3-jest TODO

Структура:

Приложение состоит из 4 шагов:

    1. Выбор общих параметров системы и формирование превью системы(Тип встройки, цвет, электропитание)
    2. Определение формы, размеров, добавление треков
    3. Отображение получившейся конфигурации и добавление светильников

Что получилось реализовать:

    1. Использовать возможности Vue3
    2. Миграция некоторых компонентов с Vue2
    3. Более глубокое понимание модулей Vuex

Что еще необходимо:

    1. Покрытие тестами
    2. Декомпозиция компонентов
    3. Рефакторинг кодовой базы, для реализации SVG рендера, которая написана "в лоб"
    4. Подключение к существующему бекенду
